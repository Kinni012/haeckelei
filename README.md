# Haeckelei

Eine Website gebaut mit [Astro](https://astro.build).

## Projektstruktur

- `src/pages/` – Seiten (jede `.astro`-Datei wird zu einer Route)
- `src/layouts/` – Wiederverwendbare Layouts
- `src/components/` – UI-Komponenten (optional anlegen)
- `public/` – Statische Dateien (Bilder, Fonts, etc.)

## Befehle

| Befehl              | Aktion                                      |
|---------------------|---------------------------------------------|
| `npm install`       | Abhängigkeiten installieren                 |
| `npm run dev`       | Dev-Server starten (z. B. http://localhost:4321) |
| `npm run build`     | Produktions-Build erstellen                 |
| `npm run preview`   | Lokale Vorschau des Builds                  |

## Nächste Schritte

- Weitere Seiten unter `src/pages/` anlegen (z. B. `about.astro` → `/about`)
- Komponenten in `src/components/` erstellen und in Seiten einbinden
- [Astro-Dokumentation](https://docs.astro.build) für Guides und API
